# ORI-POP

MANSION's personal art-making toolset and hyper-minimal creative-coding workspace in Rust. Built around the tension between order and emergence, it draws on ori — fold as structure — and pop — points in space — to explore how discrete particles organize within invisible fields.

**[Read the Guide](https://mmansion.github.io/ori-pop/)**

## Quick Start

```bash
git clone https://github.com/mmansion/ori-pop.git
cd ori-pop
cargo run -p sketches --bin hello-ori-pop
```
